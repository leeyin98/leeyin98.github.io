1. 包装类

包装类的继承结构

栗子1：

package com.leeyin98.in;
public class IntegerTest {
    public static void main(String[] args) {
        //创建一个包装类型的对象
        Integer integer1 = new Integer(1);
        System.out.println(Integer.TYPE);
        Integer integer2 = new Integer("2");
        System.out.println(integer1);
        System.out.println(integer2);
        //现在提供自动装箱和拆箱
        //自动装箱（基本数据类型-->包装类）
        Integer integer3=9;
        System.out.println(integer3);
        //自动拆箱（包装类-->基本数据类型）
        int num2 = integer3;
        String numstr = "123";
        //将字符串转换成整数
        int x = Integer.parseInt(numstr);
        System.out.println(x);
        //将字符串转换成整型的包装类
        Integer i = Integer.valueOf(numstr);
        System.out.println(i);
        System.out.println(i.intValue());
        //将一个整数进行二进制输出
        System.out.println(Integer.toBinaryString(10));
        //将一个帧数进行16进制输出
        System.out.println(Integer.toHexString(16));
    }
}

栗子2：

package com.leeyin98.in;
public class BigDeciDemo01{
  public static void main(String[] args){
    double num1 = 10;
    double num2 = 9.9;
    System.out.println(num1-num2);
    //结果为：0.0999999999964
    //问：如何解决结果才0.1
  }
}

解决方案：

package com.leeyin98.in;
import java.math.BigDecimal;

public class BigDeciDemo02 {
    public static void main(String[] args) {
        BigDecimal num1 = new BigDecimal("10");
        BigDecimal num2 = new BigDecimal("9.9");
        //减
        System.out.println(num1.subtract(num2));
        //加
        System.out.println(num1.add(num2));
        //除
        System.out.println(num1.divide(num2, 1));
        //乘
        System.out.println(num1.multiply(num2));
    }
}

3. Random

生成随机数：java.util.Random

Random r = new Random();//创建一个Random对象
for(int i = 0;i < 20;i++){//随机生成20个整数
  int num = rand.nextInt(10);//返回下一个整型的伪随机数
  System.out.println("第"+(i+1)+"个随机数是："+num);
}

注意：用同一个种子值来初始化两个Random对象，然后用每个对象调用相同的方法，得到的随机数也实现相同的

4. String

4.1 String类的基本说明

public final class String
extends Object
implements Serializable,Comparable<String>,CharSequence
  • String类代表字符串，Java程序中的所有字符串字面值(如”abc”)都作为此类的实例实现
  • 字符串是常量，它们的值在创建之后不能更改;
  • 字符串缓冲区支持可变的字符串;
  • 因为String对象是不可变的，所以可以共享

4.2 String的常用构造方法

构造方法 说明
public String(String original) 使用字符串构造一个字符串对象
public String(char[] value) 使用字符数组构造一个字符串对象
public String(char[] value,int offset,int count) 使用字符数组构造一个字符串对象
public String(byte[] bytes) 使用字节数组构造一个字符串对象
public String(byte[] bytes,int offset,int length) 使用字节数组构造一个字符串对象

4.3 String对象的创建方式

主要分为两种：

  1. 直接赋值：String str = value;

采用常量池机制保存数据，首先扫描常量池，如果发现池中没有该字符串，那么此时从常量池创建一个，如果发现池中本来就有该字符串，则直接引用即可

  1. 采用new的形式：String str = new String(value);

采用堆栈机制完成
栗子：

package com.leeyin98.string;
public class StringDemo02 {
    public static void main(String[] args) {
        String msg = "hello";
        String newMsg = new String("hello");
        String msg3 = "hello";
        String newMsg2 = new String("hello");
        //true
        System.out.println(msg.equals(newMsg));
        //false
        System.out.println(msg==newMsg);
        //true
        System.out.println(msg==msg3);
        //false
        System.out.println(newMsg==newMsg2);
    }
}

5. StringBuffer

StringBuffer类的声明

public final class StringBuffer
extends Object
implements Serializable,CharSequence

为什么要使用StringBuffer类

使用StringBuffer进行改造
创建方式只有一种：使用构造方法完成StringBuffer对象的创建操作

package com.leeyin98.string;
import java.util.Random;

public class StringDemo03 {
    public static void main(String[] args) {
        StringBuffer sb = new StringBuffer("hello");
        //String msg = "hello";
        for(int i = 0 ; i<2;i++) {
            sb.append("abcefg".charAt(new Random().nextInt("abcefg".length())));
            System.out.println(sb.toString());
            System.out.println(sb.hashCode());
        }
    }
}

当字符串内容需要频繁修改的时候，此时使用StringBuffer类会比较好，其他情况下使用String和StringBuffer基本一致

6. Object

Object类是所有类的父类

6.1 equals方法

默认实现：

public boolean equals(Object boj){
  return (this == obj);
}

==：左右两边是基本数据类型，则比较的是基本数据类型的值
==：左右两边是引用数据类型，则比较的是引用数据类型的地址值

6.2 finalize

表示Java在调用垃圾收集机制，处理垃圾之前的一个动作，在此方法中实现
栗子：

package com.leeyin98.object;

class Student{
    String name ; 
    int age ;

    public Student() {
        super();
        System.out.println("出生了");
    }
    public Student(String name, int age) {
        super();
        this.name = name;
        this.age = age;
    }
    public String getName() {
        return name;
    }
    public void setName(String name) {
        this.name = name;
    }
    public int getAge() {
        return age;
    }
    public void setAge(int age) {
        this.age = age;
    }
    @Override
    public void finalize() throws Throwable {
        //super.finalize();
        System.out.println("去世了");
    } 
}
public class Test01 {
    public static void main(String[] args) {
        Student s = new Student();
        s=null;
        System.gc();//finalize()
        //Runtime.getRuntime().gc();
    }
}

7. 日期操作

日期类：java.util.Date
构造方法：Date date = new Date()
格式化类：java.text.SimpleDateFormat
构造方法：public SimpleDateFormat(String pattern)
具体使用方法可查阅API文档
格式：

栗子：

package com.leeyin98.date;
import java.text.ParseException;
import java.text.SimpleDateFormat;
import java.util.Date;

public class DateDemo {
    public static void main(String[] args) {
        Date date = new Date();
        System.out.println(date);
        System.out.println(date.getTime());
        date.setTime(date.getTime()+24*60*60*1000);
        System.out.println(date);
        //创建SimpleDateFormat对象
        SimpleDateFormat sdf = new SimpleDateFormat("yyyy-MM-dd hh:mm:ss");
        //将date类型转换成字符串
        String str = sdf.format(date);
        System.out.println(str);
        //在将字符串转换成日期
        try {
            //将字符串转换成日期类型
            Date d=sdf.parse(str);
            System.out.println(d);
        } catch (ParseException e) {
            // TODO Auto-generated catch block
            e.printStackTrace();
        }
    }
}

8. System

方法 说明
public static void exit(int status) 退出应用程序
public static void gc() 调用垃圾收集机制
public static Properties getProperties() 取得系统的资源信息
其他方法可查阅API文档 API文档资源在笔记1中

栗子：

package com.leeyin98.system;
import java.util.Enumeration;
import java.util.Properties;

public class Test2 {
    public static void main(String[] args) {
        //Properties对象
        Properties props = System.getProperties();
        //获取props对象中所有key值
        Enumeration enums = props.keys();
        //循环读取所有的key
        while(enums.hasMoreElements()) {
            //取出当前的key值
            String key =(String)enums.nextElement();
            //getProperty(key)--->根据key取得对应的值
            System.out.println(key+"-->"+props.getProperty(key));
        }
    }
}
